# business-coach-landing-page

A resposive and modern landing page designed for coaches and educators to showcase their personal brand, attract clients, and build credibility online.

---

## Features

- Clean, semantic **HTML5** structure
- Styled with **TailwindCSS** (via CLI build)
- Responsive design design for mobile, tablet, and desktop
- Optimized assets and SEO-friendly meta tags
- Easy deployment on **GitHub pages**, **Netlify**, or **Vercel**
